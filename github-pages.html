GitHub Pages 搭建完全教程

GitHub Pages 是 GitHub 提供的免费静态站点托管服务，可以用来搭建博客、作品集、知识库等。

一、创建仓库

仓库名必须是：用户名.github.io

例如你的：pekopoke.github.io

二、上传文件

将 index.html、css、图片等静态资源直接上传到仓库 main 分支。

三、自动部署

GitHub 会自动部署，几分钟后即可访问：

https://pekopoke.github.io

四、验证网站所有权

将平台提供的验证文件放在根目录，确保可直接访问即可。
